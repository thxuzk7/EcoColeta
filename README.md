♻️ EcoColeta

O EcoColeta é um sistema simples para gestão e consulta de pontos de coleta seletiva, desenvolvido em Java utilizando uma arquitetura cliente-servidor.  
O objetivo principal é facilitar o descarte adequado de resíduos sólidos, conectando cidadãos a pontos de coleta e permitindo que administradores mantenham os dados atualizados.

🛠️ Funcionalidades

👤 Usuário
- Consultar pontos de coleta seletiva disponíveis  
- Visualizar informações formatadas sobre cada ponto  

🔑 Administrador
- Acessar painel de administração  
- Cadastrar novos pontos de coleta  
- Atualizar ou remover pontos existentes  

🔗 Arquitetura Cliente-Servidor
- Servidor: gerencia os pontos de coleta e responde às requisições  
- Cliente: interface gráfica que permite interação dos usuários e administradores  

Tecnologias Utilizadas
- Java  
- Swing (javax.swing, java.awt) → Interfaces gráficas  
- Sockets (java.net) → Comunicação cliente-servidor  
- I/O Streams (java.io) → Entrada e saída de dados entre cliente e servidor
